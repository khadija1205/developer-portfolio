# Animated Portfolio

A modern, interactive portfolio website showcasing projects and skills with smooth animations and a sleek dark theme.

##  Features

-  Dark theme with vibrant purple gradient accents
-  Dynamic rotating job titles on hero section
-  Smooth animations and transitions using Framer Motion
-  Fully responsive design across all devices
-  Integrated contact form
-  Clean and minimal user interface
-  Project showcase with categorized work

##  Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

First, run the development server:
bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load fonts.
